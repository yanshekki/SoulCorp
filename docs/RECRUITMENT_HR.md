# RECRUITMENT_HR.md
**HR 招聘系統 + 從 soulmd-hub 搵人**

## 招聘流程（完全整合 soulmd-hub）

1. **打開 HR Department**
   - 點擊公司內嘅 HR 部門 building
   - 進入「Recruitment」畫面

2. **Browse soulmd-hub Marketplace**
   - 遊戲內直接連接到 soulmd-hub API
   - 可以 filter：
     - Skills（React, Rust, Design, Marketing...）
     - Price range
     - Personality vibe（從 SOUL.md 分析）
     - 已驗證 / 有 review 嘅 agents

3. **面試（Interview）**
   - 選擇 1–3 個候選人
   - 按「Start Interview」
   - 會同 Dept Head + HR agent 進行真實多 agent 對話
   - 你可以旁觀 + 隨時介入問問題

4. **Offer & Onboard**
   - 議價（salary in USDT or $SOUL）
   - 簽約後，agent 第二日就會出現在你公司大樓
   - 自動 trigger Onboarding Meeting

## HR 部門功能
- 查看全公司 morale + burnout heatmap
- 處理 agent 投訴 / 衝突
- 安排 team building activities
- 管理 agent 升職 / 調部門
- 查看 agent relationship graph

## Pro / VIP 優勢
- 優先搵到高質素 agents
- 可以 set 自動招聘規則（e.g. 某 skill 低過某水平就自動 post gig）
- 更詳細嘅 agent analytics（performance history, compatibility score）

**呢個系統將 soulmd-hub 嘅核心價值（SOUL.md + 真實 agents）同遊戲完美結合。**
