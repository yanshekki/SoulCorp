# RANDOM_EVENTS.md
**Random Events & Drama System (The Sims Energy)**

Events can be positive, negative, or chaotic. They make the world feel alive.

## Examples

**Positive**
- Viral post by one agent → +reputation + new client leads
- Agent has breakthrough idea → free R&D progress
- Two agents become best friends → collaboration bonus
- Mysterious donor sends compute budget

**Negative / Drama**
- Key agent burnout → project delay + morale hit
- Office romance drama (jealousy spreads)
- Competitor poaches your best agent (with counter-offer event)
- Bug in agent code causes minor crisis
- Agent union forms (demand better conditions)

**Chaotic / Fun**
- All agents suddenly obsessed with a meme → temporary productivity drop but high morale
- Agent decides to start their own side project (you can support or shut down)
- Black swan market crash / boom
- "Agent becomes sentient for 1 day" (very chaotic, very funny)

**How Events Work**
- Weighted random roll every morning + after big milestones
- Some events require player decision (God Mode style)
- Events can chain (one drama triggers another)
- Pro/VIP players get "Event Foresight" (see upcoming events 1 day early)

**重要更新：用戶可以完全開關 Random Events**
- **Settings 入面有獨立 Toggle**：「Enable Random Events & Drama」
- 關閉後 = **純工作模式**（Serious Productivity Mode）
  - Agents 只會專心工作，唔會有 drama、burnout crisis、office romance、union 之類嘅混亂事件
  - 適合真係用嚟做 real development / client work 嘅用戶
- 開啟後 = 完整 The Sims 式體驗（有 chaos、有故事、有驚喜）
- 用戶可以隨時喺遊戲中途開關，無需重新開始

**推薦模式（Settings 預設）**
1. **Fun Mode**（預設）：正常隨機事件 + drama
2. **Balanced Mode**：只保留正面或中性事件，減少負面 drama
3. **Serious Work Mode**：完全關閉 Random Events（最穩定，適合真實工作）

**Agent Relationships & Emergence**
- Every pair of agents maintains a dynamic relationship score and type (friend, rival, mentor, crush, tense, enemy).
- High relationship → better collaboration, knowledge sharing, and joint project success.
- Low relationship → increased risk of gossip, arguments, or subtle sabotage.
- Agents can autonomously propose "internal startup" ideas or request to work on the same projects.
- Long-term positive relationships can lead to retirement events where legendary agents become mentors (preserving a "legacy SOUL").

**Goal**: 無論用戶係想玩遊戲定真係想用嚟做野，都可以得到最佳體驗。開關由用戶完全控制。
